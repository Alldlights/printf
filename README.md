C - printf

Simple implementation of printf function that formats and prints data

Description
This fucntion produces output according to a format specifier. writes output to the stdout.

Format sepcifiers:
* %c: prints a single character
* %s: print string of characters
* %d: prints a decimal integer
* %i: prints an integer
* %b: prints the binary representation of an unsigned decimal
* %u: prints an unisgned integer
* %x: prints the hexadecimal representation of an unsigned decimal in lower case
* %X: prints the hexadecimal representation of an unsigned integer in uppercase letter
* %r: prints a reversed string
* %R: prints the Rot13 interpretation of a string
